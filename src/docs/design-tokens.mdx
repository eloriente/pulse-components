# Design Tokens

Pulse Components uses **CSS custom properties** (`--pulse-*`) as the single source of
truth for design. All component styles reference these variables internally, making
consistent theming and brand updates as simple as overriding a few values.

## Quick Start

Import the token stylesheet to use all available tokens:

```css
@import 'pulse-components/tokens';
```

Then use tokens in your CSS:

```css
.my-component {
  color: var(--pulse-color-text);
  background: var(--pulse-color-background);
  padding: var(--pulse-space-4);
  font-size: var(--pulse-font-size-md);
}
```

---

## Using Semantic Tokens

**Always use semantic tokens**, not the raw neutral scales. Semantic tokens are contextual
aliases that make your code intention-clear and make theming updates automatic.

For example:
- ✅ Use `var(--pulse-color-text)` for body copy
- ❌ Avoid `var(--pulse-color-neutral-800)`

When you override semantic tokens in `:root`, all components update automatically:

```css
:root {
  --pulse-color-text: #1a1a1a;
  --pulse-color-background: #f9f9f9;
}
```

## Complete Example

Here's how to compose semantic tokens into a real component:

```css
.card {
  background-color: var(--pulse-color-background);
  border: var(--pulse-border-width) solid var(--pulse-color-border);
  border-radius: var(--pulse-radius-md);
  padding: var(--pulse-space-6);
  font-family: var(--pulse-font-family-base);
  font-size: var(--pulse-font-size-md);
  line-height: var(--pulse-line-height-normal);
  color: var(--pulse-color-text);
}

.card-title {
  font-size: var(--pulse-font-size-lg);
  font-weight: var(--pulse-font-weight-bold);
  margin-bottom: var(--pulse-space-4);
}

.card-subtitle {
  font-size: var(--pulse-font-size-sm);
  color: var(--pulse-color-text-subtle);
}
```

## Theming

Override semantic tokens in `:root` to change your app's appearance globally:

```css
:root {
  /* Dark mode */
  --pulse-color-text: #f5f5f5;
  --pulse-color-text-subtle: #b0b0b0;
  --pulse-color-background: #1a1a1a;
  --pulse-color-border: rgba(255, 255, 255, 0.1);

  /* Brand colors */
  --pulse-color-primary: #7c3aed;
  --pulse-color-primary-dark: #6d28d9;
  --pulse-color-primary-light: #a78bfa;
}
```

All components using semantic tokens will automatically reflect these changes.
