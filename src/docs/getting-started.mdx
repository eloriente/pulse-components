# Getting Started

Everything you need to install Pulse Components and render your first component.

---

## Prerequisites

- **Node.js** >= 18
- **npm** >= 9
- **React** >= 18 (peer dependency)

---

## Installation

```bash
npm install pulse-components
```

---

## Importing styles

Pulse Components ships its own CSS. Import it **once** in your application entry point
(e.g. `main.tsx`, `_app.tsx`, `layout.tsx`):

```tsx
import 'pulse-components/styles';
```

If you also want access to the raw `--pulse-*` CSS custom properties in your own
stylesheets, import the tokens separately:

```css
@import 'pulse-components/tokens';
```

> You do **not** need both. `pulse-components/styles` already includes the component
> styles. Use `pulse-components/tokens` only when you want to reference the variables
> directly in your own CSS files.

## Your first component

### Button

```tsx
import { Button } from 'pulse-components';
import 'pulse-components/styles';

export default function App() {
  return (
    <div>
      <Button label="Log in" onClick={() => console.log('login')} />
      <Button primary label="Sign up" onClick={() => console.log('signup')} />
    </div>
  );
}
```

### Header

```tsx
import { Header } from 'pulse-components';
import 'pulse-components/styles';

export default function App() {
  const [user, setUser] = React.useState<{ name: string }>();

  return (
    <Header
      user={user}
      onLogin={() => setUser({ name: 'Jane Doe' })}
      onLogout={() => setUser(undefined)}
      onCreateAccount={() => setUser({ name: 'Jane Doe' })}
    />
  );
}
```

**Header props**

| Prop | Type | Description |
|---|---|---|
| `user` | `{ name: string }` | When provided, shows a welcome message and a Log out button |
| `onLogin` | `() => void` | Called when the user clicks "Log in" |
| `onLogout` | `() => void` | Called when the user clicks "Log out" |
| `onCreateAccount` | `() => void` | Called when the user clicks "Sign up" |
| `className` | `string` | Extra CSS classes applied to the root `<header>` element |

---

### Page

`Page` is a full-page composition demo that combines `Header` with a content section.
It manages its own internal `user` state.

```tsx
import { Page } from 'pulse-components';
import 'pulse-components/styles';

export default function App() {
  return <Page />;
}
```

**Page props**

| Prop | Type | Description |
|---|---|---|
| `className` | `string` | Extra CSS classes applied to the content `<section>` |

---

## Package exports

The package exposes three entry points:

| Import path | What it provides |
|---|---|
| `pulse-components` | All components and TypeScript types |
| `pulse-components/styles` | Bundled component CSS (`dist/style.css`) |
| `pulse-components/tokens` | Design token CSS variables only (`dist/tokens/index.css`) |

---

## TypeScript

All prop interfaces are exported and can be imported for use in your own code:

```tsx
import type { ButtonProps, HeaderProps, PageProps } from 'pulse-components';
```
