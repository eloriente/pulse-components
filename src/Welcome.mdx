# Pulse Components

An open-source React component library built with **TypeScript** and **CSS Modules**.
Designed to be used as-is, forked, or extended to match your own design system.

---

## What is Pulse Components?

Pulse Components is a lightweight, accessible UI kit that gives you production-ready
building blocks without locking you into a heavy framework. Every component:

- Is fully typed with **TypeScript** interfaces
- Uses **CSS Modules** — no class-name collisions, ever
- Consumes **`--pulse-*` design tokens** that you can override globally
- Accepts a `className` prop for **Tailwind CSS** or any utility-first workflow
- Is tested in a real browser via **Vitest + Playwright**

## Quick install

```bash
npm install pulse-components
```

```tsx
import { Button } from 'pulse-components';
import 'pulse-components/styles';

export default function App() {
  return <Button primary label="Get started" />;
}
```
